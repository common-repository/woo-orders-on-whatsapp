== Description ==

Woocommerce Orders on Whatsapp allows your customers to contact you and chat via Whatsapp directly from your wordpress/woocommerce products pages to the mobile.

== Installation ==

1. Upload the plugin files to the '/wp-content/plugins/xc-woo-whatsapp-order' directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Plugin Name screen to configure the plugin
